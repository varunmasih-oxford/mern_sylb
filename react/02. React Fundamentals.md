# React Fundamentals

## 1. What is React?

React is a **JavaScript library for building user interfaces (UI)**, especially for web applications.

It helps developers create **interactive, fast, and reusable UI components**.

### Key Features of React
- Component-based architecture
- Fast rendering using Virtual DOM
- Reusable UI components
- Strong community and ecosystem

---

## 2. Single Page Application (SPA)

A **Single Page Application** loads **one HTML page** and updates content dynamically without refreshing the browser.

### Traditional Website
- Each click loads a new page from the server

### SPA (React Apps)
- Only one page loads
- Content changes dynamically using JavaScript
- Faster and smoother user experience

Examples of SPAs: Gmail, Facebook, Instagram

---

## 3. Virtual DOM

The **DOM (Document Object Model)** is the browser’s structure of a webpage.

Updating the real DOM is slow. React solves this using a **Virtual DOM**.

### How Virtual DOM Works
1. React creates a virtual copy of the real DOM
2. When state changes, React updates the Virtual DOM first
3. React compares old and new Virtual DOM (Diffing)
4. Only the changed parts are updated in the real DOM

### Benefits
- Faster updates
- Better performance
- Efficient UI rendering

---

## 4. Setting Up React

React projects are commonly created using **Vite** or **Create React App (CRA)**.

### Option 1: Using Vite (Recommended)

```bash
npm create vite@latest my-react-app
cd my-react-app
npm install
npm run dev
````

### Option 2: Using Create React App

```bash
npx create-react-app my-react-app
cd my-react-app
npm start
```

After running the command, open the browser at:

```
http://localhost:5173   (Vite)
http://localhost:3000   (CRA)
```

## 5. Understanding React Project Structure

Here’s a basic folder structure of a React project:

```
my-react-app/
│
├── node_modules/      # Installed packages
├── public/            # Static files
│   └── index.html     # Main HTML file
│
├── src/               # Main working folder
│   ├── App.jsx        # Main App component
│   ├── main.jsx       # Entry point of React app
│   ├── components/    # Custom components (you create)
│   └── assets/        # Images, styles, etc.
│
├── package.json       # Project details & dependencies
└── vite.config.js     # Vite configuration (if using Vite)
```

### Important Files

**index.html**

* The root HTML file
* Contains a `<div id="root"></div>` where React loads

**main.jsx**

* Entry point of the React app
* Renders the main `App` component

```js
import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App";

ReactDOM.createRoot(document.getElementById("root")).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);
```

**App.jsx**

* Main component where your UI starts

```js
function App() {
  return <h1>Hello React</h1>;
}

export default App;
```

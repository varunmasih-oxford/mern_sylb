# JSX (JavaScript XML)

JSX stands for **JavaScript XML**. It allows us to write HTML-like syntax inside JavaScript and is used in React to describe the user interface.

---

## 1. Writing HTML Inside JavaScript

In React, we use JSX to create UI elements.

```jsx
const element = <h1>Hello, React!</h1>;
````

JSX looks like HTML, but it is actually converted into JavaScript behind the scenes.

You can return JSX from a component:

```jsx
function App() {
  return <h1>Welcome to React</h1>;
}
```

---

## 2. JSX Rules

JSX must follow certain rules to work correctly.

### Rule 1: Single Parent Element

JSX must return only **one parent element**.

❌ Incorrect:

```jsx
return (
  <h1>Hello</h1>
  <p>Welcome</p>
);
```

✅ Correct:

```jsx
return (
  <div>
    <h1>Hello</h1>
    <p>Welcome</p>
  </div>
);
```

You can also use a React Fragment:

```jsx
return (
  <>
    <h1>Hello</h1>
    <p>Welcome</p>
  </>
);
```

---

### Rule 2: All Tags Must Be Closed

Every tag must be properly closed.

```jsx
<img src="image.jpg" />   // Correct
<input type="text" />     // Correct
<br />                    // Correct
```

---

## 3. JavaScript Expressions in JSX `{}`

You can write JavaScript expressions inside JSX using curly braces `{}`.

```jsx
const name = "Varun";

<h1>Hello {name}</h1>
```

You can also use calculations:

```jsx
<p>2 + 2 = {2 + 2}</p>
```

Or variables and functions:

```jsx
const greet = () => "Good Morning";

<h2>{greet()}</h2>
```

---

## 4. Inline Styling in JSX

Inline styles are written as JavaScript objects.

```jsx
const style = {
  color: "blue",
  fontSize: "20px"
};

<h1 style={style}>Styled Text</h1>
```

You can also write styles directly:

```jsx
<h1 style={{ color: "red", backgroundColor: "yellow" }}>
  Hello
</h1>
```

Note: CSS property names use **camelCase** (for example, `backgroundColor` instead of `background-color`).

---

## 5. className vs class

In HTML, we use the `class` attribute.
In JSX, we must use **`className`** because `class` is a reserved keyword in JavaScript.

```jsx
<h1 className="title">Hello React</h1>
```
